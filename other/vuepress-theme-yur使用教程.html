# 添加分类

  • 在 /docs 下新建一个文件夹 other：一个文件夹代表一个分类
  • 在 /docs/other 下新建一个文件 README.md：代表该分类的列表页

README.md 内容如下：

---

title: 分类名称

meta:

  - name: description

    content: 分类描述

  - name: keywords

    content: 分类关键词

---

update_date: 2019-09-12
