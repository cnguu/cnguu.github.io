# 注意事项

# 主题破解补丁

# 安装

  1. 右键，以管理员身份运行
  2. Next
  3. 勾选同意条款，Next
  4. I Agree
  5. 记住当前 Themeui 和 Uxinit 的 Status
  6. Install
  7. Next
  8. Next
  9. 重启系统，Finsh

# 验证

重新打开安装程序，查看 Status 是否都是 patched

是则破解成功

否则，需要取得管理员所得权

# 管理员所得权

  1. 运行“添加管理员取得所有权到右键菜单.reg”
  2. 打开 C:\Windows\System32
  3. 搜索 Themeui.dll，右键，管理员取得所有权
  4. 同理搜添加 Uxinit.dll
  5. 重新执行安装

# 资源管理器调整工具

# 安装

  1. 解压 OldNewExplorer 文件夹至 C 盘根目录
  2. 右键，以管理员身份运行 OldNewExplorerCfg.exe
  3. 仅勾选以下选项
    • 使用经典驱动器分组
    • 使用库、并隐藏顶部文件夹
    • 使用工具栏代替功能区
    • 隐藏标题栏文字
    • 隐藏标题栏小图标
    • 隐藏返回上级目录按钮
  4. 安装

# 主题

  1. 以蕾丝主题为例，直接点击下一步安装 lovelace_TW10.exe
  2. 快捷键 Win+i 打开系统设置 >> 个性化 >> 主题
  3. 点击其中一个主题：Pacman No Address Bar
  4. 快捷键 Win+e 打开文件资源管理器 >> 工具 >> 文件夹选项 >> 查看 >> 取消勾选：始终显示菜单

# 开始菜单调整工具

# 安装

  1. 右键，以管理员身份运行 StartIsBack2.9.exe
  2. 为所有人安装

# 配置

  1. 右键开始菜单 >> 属性
  2. 外观 >> 视觉样式：lovelace NA + Windows10 按钮
  3. 自定义外观 >> 添加
  4. 勾选：使用自定义开始菜单颜色 >> 透明度 0% >> 清除模糊
  5. 勾选：使用自定义任务栏颜色 >> 透明度 0% >> 清除模糊
  6. 勾选：隐藏用户账号图像
  7. 自定义任务栏特效 >> 勾选：任务栏图标居中
  8. 切换 >> 任务栏和开始菜单的上下文菜单样式：经典

# 图标

  1. 打开 7tsp GUI v0.6(2019).exe
  2. Add a Custom Pack：7TSP Lumicons.7z
  3. Start Patchin（约 3 分钟）
  4. 重启系统

# 托盘时间

  1. 安装字体 DS-DIGI.TTF
  2. 运行 mytime.exe
  3. 设置 >> 显示设置 >> 设置字体 >> 颜色：水绿色

# 网速监控

  1. 运行 TrafficMonitor.exe
  2. 右键 >> 显示任务栏窗口、显示主窗口
  3. 右键 >> 显示通知区域图标、显示 CPU 和内存利用率
  4. 配置任务栏窗口
    • 背景颜色选取任务栏
    • 透明色选取任务栏
    • 内存改为 RAM
    • 勾选：网速显示简洁模式
    • 勾选：水平排列
    • 勾选：任务栏窗口显示在任务栏的左侧
    • 勾选：指定每个项目颜色
    • 文本颜色
      • 上传 - 标签：#57c0ff - 数值：#ff57ff
      • 下载 - 标签：#ff8057 - 数值：#00ff00
      • CPU - 标签：#8057ff - 数值：#ffc000
      • 内存 - 标签：#00ffff - 数值：#a0ffc0
